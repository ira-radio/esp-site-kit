# espace Unit Kit

**espace Unit Kit** は、個人サイト向けの軽量で使いやすい Web パーツセットです。  
HTML・CSS・JavaScript で動作するユニット（パーツ要素）を提供し、サイト制作を軽やかにサポートします。

[🚀 公式サイト (デモ・ドキュメント)](https://espace-uk.i-ra.site/)

---

## ✨ 特徴

- **軽量 & 高速**: 必要な機能を厳選し、サイトのパフォーマンスを損なわない設計。
- **簡単導入**: 複雑な環境構築は不要。HTML/CSS/JS を読み込むだけで動作します。
- **柔軟なカスタマイズ**: 個人サイトの雰囲気に合わせて調整しやすい設計。

## 🚀 導入方法

詳細な導入ガイドや各ユニットの使い方は、[公式サイトの導入マニュアル](https://espace-uk.i-ra.site/)をご覧ください。

### 基本的な読み込み

ダウンロードしたファイルをプロジェクトに配置し、以下のように読み込みます。

```html
<html>
    <head>
        <!-- espkit: CSS読み込み -->
        <link
            rel="stylesheet"
            href="path/to/espkit-all.min.css"
        />
    </head>
    <body>
        <!-- コンテンツ -->

        <!-- espkit: JavaScriptの読み込み（JavaScriptが必要な機能を使用する場合） -->
        <script src="path/to/espkit-all.min.js"></script>
    </body>
</html>
```
